Построить таблицы умножения в конечном поле, необходимые для реализации алгоритма шифрования «Кузнечик».
• Реализовать метод шифрования/расшифрования блока алгоритма «Кузнечик» в соответствии с ГОСТ 34.12.
• Реализовать режимы шифрования в соответствии с ГОСТ 34.13
• Расшифруйте данные из файла 12.kuznyechik. Ключ и данные представлены в base16 и base64 форматах соответственно (использовался режим ECB).
• Линейное преобразование L шифра «Кузнечик» действует на векторном пространстве (𝔽₂⁸)¹⁶, т.е. L ∈ GL₁₆(𝔽₂⁸). Найдите его матричное представление M и реализуйте его вычисление с использованием этой матрицы, т.е. L(x) = M·x
